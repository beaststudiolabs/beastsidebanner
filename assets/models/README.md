# 3D Character Models

Place GLB character models here:
- boy1.glb (2 boys)
- boy2.glb
- girl1.glb (3 girls)
- girl2.glb
- girl3.glb

## Requirements
- Format: GLB (binary glTF)
- Max size: 3MB each
- Morph targets: Named to ARKit blendshape standard
- Minimum 20 core expressions (eyes, mouth, brows)
- Optional: Full 52 ARKit blendshapes for maximum expressiveness

## ARKit Blendshape Names
Eye: eyeBlinkLeft, eyeBlinkRight, eyeWideLeft, eyeWideRight
Mouth: jawOpen, mouthSmileLeft, mouthSmileRight, mouthFrownLeft, mouthFrownRight
Jaw: jawLeft, jawRight, jawForward
Brow: browInnerUp, browOuterUpLeft, browOuterUpRight

Full list: https://developer.apple.com/documentation/arkit/arfaceanchor/blendshapelocation
